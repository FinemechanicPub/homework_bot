# Телеграм-бот для работы с внешним сервисом

## Установка

Создание и активация окружения
```bash
python -m venv venv
source venv/bin/activate
```
Установка зависимостей
```bash
pip install -r requirements.txt
```

## Использование

Бот запускается командой

Установка зависимостей
```bash
pip homework.py
```
